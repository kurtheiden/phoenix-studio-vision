# Local samples

This directory is a local workspace for researching Studio Vision projects.
Its sample files and category-directory contents are ignored by Git by default.

Do not commit personal, copyrighted, confidential, or non-redistributable
Studio Vision projects. Only synthetic fixtures or files whose owners have
given explicit redistribution permission may eventually be committed, after a
deliberate review of their contents and rights.

Original files must never be modified. Preserve an untouched original and use
a separate working copy for inspection or future recovery experiments.

## Local categories

- `midi-only/`: projects known or believed to contain MIDI only
- `audio-linked/`: projects that reference external audio
- `mixed/`: projects containing both MIDI and audio material
- `damaged/`: incomplete, corrupted, or otherwise damaged projects
- `missing-extension/`: project files without a filename extension

## Sample records

For every local sample, copy `sample-notes-template.md` into the sample's local
folder and record:

- its provenance and the basis for permission to use it;
- the known Studio Vision or Studio Vision Pro version;
- whether it contains MIDI, audio, or both;
- whether it still opens in the original software; and
- any missing media.

Completed notes remain local by default because category-directory contents
are ignored. Do not place sensitive information in filenames or notes.
